# NBA Player Stats App

URLにNBA選手のフルネームを入力すると、その選手のシーズンごとの平均スタッツを表示するアプリです。
（例: URLに "stephen curry" と入力。大文字小文字は区別せず、姓と名の間にスペースが必要）

## 使用技術
- Python
- FastAPI
- NBA API
- SQLAlchemy (DB操作用ORM)

## 工夫した点

### 1. DBキャッシュによるタイムアウト対策
当初、全選手のスタッツを一括でDBに保存しようとしたところ、NBA APIへのリクエスト量が多すぎてタイムアウトが発生した。
対策として、リクエストが来た選手のデータのみをその都度取得する方式に変更。
まずDBを確認し、データがなければNBA APIを叩いて取得・保存する。
選手のスタッツは多くても10行程度のデータ量のため、API経由でもタイムラグはほぼなく、2回目以降のリクエストはDBから直接返すことでさらに高速化している。

### 2. timedeltaによるデータ更新
NBAのスタッツは更新されるが、一度DBに保存したデータはそのままでは更新されない問題があった。
そこでtimedeltaを使い、DBに保存された日時から1日以上経過している場合は、NBA APIから最新のスタッツを再取得してDBを更新するようにした。
これにより、キャッシュの高速性を保ちながら、データを一定期間ごとに最新にする設計にしている。
